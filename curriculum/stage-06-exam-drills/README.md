# Stage 06 - Exam Drills

## 목표

- 실제 시험 스타일의 장문 시나리오를 빠르게 해석한다.
- 정답 선택보다 오답 제거 근거를 명확히 말할 수 있다.

## 풀이 규칙

1. 문제에서 요구하는 최우선 목표(비용, 가용성, 운영복잡도)를 먼저 표시
2. 오답 보기는 "요구사항 불일치" 근거로 제거
3. 정답 확인 후 꼬리질문까지 말로 복기

## 연계 문제

- 객관식 실전: Q19 ~ Q27
- TXT 실전 드릴: `content/questions/exam_txt_drills.json` (원문 기반 자동 확장)
- 모의고사: `web/index.html` Mock Exam 섹션

## 출처 연결

- 실전 시나리오 원문: `references/raw/saa-c03-solution.txt`
- 인덱스: `references/indexes/txt-question-index.md`
- 요약: `references/indexes/txt-drill-summary.md`

## 완료 기준

- 모의고사 20문항 기준 72% 이상
- 취약 도메인(보안/회복탄력성/고성능/비용)별 개선 메모 작성 완료
