# AWS SAA Curriculum

이 폴더는 초보자부터 SAA 실전 수준까지 단계별 학습 경로를 제공합니다.

## Stage Map

- `stage-01-foundations`: 클라우드/AWS 기본 개념 이해
- `stage-02-network-security`: IAM, VPC, 네트워크/보안 설계
- `stage-03-storage-database`: 스토리지/데이터베이스 선택 기준
- `stage-04-resilience-performance`: 고가용성/확장성/성능 설계
- `stage-05-cost-operations`: 비용 최적화와 운영 관점
- `stage-06-exam-drills`: 실전 시나리오 문제 풀이 및 오답 교정
