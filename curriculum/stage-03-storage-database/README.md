# Stage 03 - Storage and Database

## 목표

- 저장소 유형(객체/블록/파일)에 맞는 서비스를 선택할 수 있다.
- 관계형/NoSQL DB 선택 시 기준을 설명할 수 있다.

## 핵심 용어

- S3 / EBS / EFS
- RDS Multi-AZ / Read Replica
- DynamoDB Capacity Mode
- Backup and Restore
